# Life Expectancy Prediction using ANN

This project uses an Artificial Neural Network (ANN) to predict life expectancy using WHO dataset.

## Technologies Used
- Python
- TensorFlow / Keras
- Pandas
- Scikit-learn

## Project Steps
- Data preprocessing
- Handling missing values
- Feature scaling
- Train-test split
- ANN model building
- Model evaluation

## Evaluation Metrics
- MAE
- MSE
- R² Score
